Gitboard by KuiKui

Simple dashboard for a quick overview of Git projects.

Screenshot

Usage

php gitboard.phar
or add this line to your git config file : 
[alias]
board = "!f() { php /path/to/gitboard/gitboard.phar $@; }; f"
and simply use : 
git board
you can convert ansi output to html : 
php gitboard.phar --display-web > gitboard.html

Options

  • -d : like --git-dir
  • -i : number of last days/hours/minutes
  • -c : number of last commits
  • -h : help
  • -v : version
  • --no-merged-branch : no merged branches infos
  • --no-stat : no statistic
